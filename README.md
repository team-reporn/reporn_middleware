# Reporn Node Server

## 🏃 Run the project

### Requirements
Have node.js

### Clone the repository :

    git clone https://github.com/team-reporn/reporn_middleware.git
    
### Setup the project :

Install packages :

    npm install
    
Configure API adresse :

    schema.js
    
    33. const url = "http://api.reporn.remiruc.com/api/users";
    
## Launch the project

    node serve.js
